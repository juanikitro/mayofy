# google-business-validation

Valida datos obtenidos desde fuentes permitidas.

No scrapea agresivamente. Usa APIs oficiales, exports autorizados o revision manual con URLs publicas.

Salida minima:

- campos verificados
- fuentes
- fecha de captura
- datos faltantes
- apto o no apto para generacion

Comando inicial:

```bash
npm run search:tandil
npm run validate:intake
```
